## conda 创建环境、删除环境
``` python
conda create -n py36 python=3.6 
conda remove -n py36 --all
```

## 查看cuda
``` python
nvidia-smi
```

## 查看gpu占用
``` python
watch --color -n1 gpustat -cpu
```

## 安装pytorch
``` python 
conda install pytorch torchvision torchaudio cudatoolkit=11.6 -c pytorch -c conda-forge
```
